## 简介

参考 [vue-vben-admin](https://github.com/vbenjs/vue-vben-admin)，使用了最新的 Vue3 + Vite2 + Element-plus 技术栈搭建的简易版后台模板。

## 安装使用

- 安装依赖

```sh
cd vue3-admin-template
pnpm i
```

- 运行

```sh
pnpm serve
```

- 打包

```sh
pnpm build
```
